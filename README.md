# 今日のスケジュールを送るslackbot

## 問題  
- 1日のスケジュールを把握できていない
- 予定を忘れてしまう

## 解決策  
決まった時間に1日のスケジュールを送る  

## メモ  
### PropertiesService  
Google apps scriptの機能  
コード上に機密情報などを残したくない時に使う  
### JSONオブジェクト  
jsonを解釈するメソッドや、値をjsonに変換するメソッドを持つ  
### UrlFetchApp  
Google apps scriptの機能  
他のアプリケーションと通信したり、ウェブ上の他のリソースにアクセスするのに使う  
### CalendarApp  
Google apps scriptの機能  
Googleカレンダーの読み取りや更新をするのに使う
